# 🎓 College Website – Modern Web Application

🚀 **Live Demo** https://college-website-liard.vercel.app/

A fully responsive and dynamic **College Website** that provides complete details about the institution, courses (from **LKG to 12th** in **Arts**, **Science**, and **Commerce**), online admission registration, and secure online fee payment using **Razorpay**.  
The platform also includes **Google Sign-In authentication** powered by **Auth0** and is hosted on **Render**.

---

## ✨ Key Features

- 🏫 **College Overview** – Complete information about the college, departments, and facilities.  
- 📚 **Courses** – Structured course listing from **LKG to 12th grade** under **Arts**, **Science**, and **Commerce** streams.  
- 🧾 **Online Admission** – Simple and secure student admission form with validation.  
- 💳 **Online Fee Payment** – Integrated **Razorpay** gateway for safe and easy transactions.  
- 🔐 **Authentication** – Seamless **Google Sign-In** using **Auth0**.  
- 🌐 **Deployed on Render** – Optimized for performance and scalability.  
- 📱 **Fully Responsive UI** – Works smoothly across desktops, tablets, and smartphones.
